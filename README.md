# 文化遗产数字体验系统

专为文化馆、博物馆设计的互动式人脸融合体验应用，让游客能够与珍贵的历史文物、古代雕像进行数字化融合，创造独特的文化纪念体验。

## 🏛️ 项目特色

### 文化教育价值
- **历史文化传承** - 通过科技手段让历史文物"活"起来
- **沉浸式体验** - 游客可以"穿越"到历史场景中
- **文化认知提升** - 增强对传统文化的理解和兴趣
- **数字化保护** - 以数字方式传播和保护文化遗产

### 技术特性
- 🎭 **智能人脸融合** - AI驱动的高质量面部特征融合
- 🎨 **风格保持技术** - 完美保持文物的历史质感和艺术风格
- 📱 **响应式设计** - 适配各种展示设备和触摸屏
- 🖼️ **专业级输出** - 高分辨率结果图片，适合打印收藏
- 📱 **二维码分享** - 自动生成二维码，游客可扫码下载图片
- 🔗 **多种下载方式** - 直接下载、扫码下载、分享链接
- 💾 **智能存储** - 24小时临时存储，保护用户隐私

## 🎯 核心功能

### 二维码下载系统
- **自动生成二维码** - 每张融合图片都会生成专属下载二维码
- **扫码即下载** - 游客用手机扫码即可获取高清图片
- **专属下载页面** - 提供详细的图片信息和多种下载选项
- **24小时有效期** - 保护隐私的同时提供充足下载时间

### 多种分享方式
1. **直接下载** - 点击按钮立即下载到设备
2. **二维码分享** - 扫码获取，方便手机用户
3. **链接分享** - 生成专属链接，可发送给朋友
4. **社交媒体** - 一键分享到微信、微博等平台

### 用户体验流程
1. 上传个人照片和选择文物图片
2. AI智能融合，保持文物质感
3. 生成带二维码的高质量图片
4. 多种方式下载和分享纪念照

## 🚀 快速部署

### 环境要求
- Node.js 16+ 
- 现代浏览器支持
- 触摸屏设备（推荐）

### 安装步骤
```bash

git clone [repository-url]
cd cultural-heritage-experience

# 安装依赖

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

```

## 🔧 AI服务集成

### 推荐AI服务商

#### 1. 火山引擎（字节跳动）
- **优势**: 人脸融合效果优秀，支持风格保持
- **文档**: https://www.volcengine.com/docs/6791/97889
- **适合场景**: 高质量文化体验需求

#### 2. 百度AI开放平台
- **优势**: 中文支持好，价格相对便宜
- **文档**: https://ai.baidu.com/ai-doc/FACE/ek37c1qiz
- **适合场景**: 中小型文化馆预算有限

#### 3. 腾讯云人脸融合
- **优势**: 稳定性好，并发处理能力强
- **文档**: https://cloud.tencent.com/document/product/670/31061
- **适合场景**: 大型博物馆高并发需求

### 集成配置示例

```javascript
// 编辑 src/api/faceFusion.js
import { API_CONFIG } from './config.js'

// 选择使用的AI服务
const currentAPI = new VolcanoAPI() // 或 BaiduAPI, TencentAPI

// 配置API密钥
currentAPI.apiKey = 'your-api-key'
currentAPI.secretKey = 'your-secret-key'
```

## 🛡️ 隐私保护

### 数据安全措施
- **本地处理优先** - 图片处理尽量在本地完成
- **临时存储** - 上传图片仅临时存储，处理完成后自动删除
- **用户同意** - 明确的隐私政策和使用协议
- **数据加密** - API传输使用HTTPS加密

### 合规建议
- 在显著位置展示隐私政策
- 获得用户明确同意后再处理图片
- 定期清理临时文件
- 遵守当地数据保护法规
